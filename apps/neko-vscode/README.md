# OpenNeko for VSCode

OpenNeko for VSCode is the canonical OpenNeko Extension Pack product. It contains
product metadata only and installs the domain Extensions listed by
`extensionPack`.

- Product identity: `neko.neko-suite`
- Product build/package root: `apps/neko-vscode`
- Domain Extensions and Custom Editors: owned by their respective
  `packages/neko-*` packages

This application root must not contain Extension runtime, commands, providers,
Webviews, Custom Editors, or domain implementations.
